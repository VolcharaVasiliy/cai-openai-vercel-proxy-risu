Risu Setup

  1. Open Risu settings and set AI model to reverse_proxy.
  2. Set URL to (base URL only).
  3. Set Key/Password: if server has CAI_TOKEN, key can be empty; otherwise use your c.ai token.
  4. Set Request Model to one alias from /v1/models (example: cai-default).
  5. Set Format to OpenAI Compatible.
  6. Keep Autofill Request URL enabled (Risu appends /v1/chat/completions automatically).
Risu already manages chat history on its side; this proxy accepts normal OpenAI message arrays.
